# INTERPOLATION AND APPROXIMATION OF THE REAL PROPERTY PRICE INDEX (RPPI) IN MAKATI CITY USING NUMERICAL METHODS
  
Numerical analysis techniques to analyze and estimate the Real Property Price Index (RPPI) for Makati City. 
This study applied numerical analysis techniques to analyze RPPI data for Makati City. Linear interpolation, lagrange polynomial, and cubic spline interpolation were used to estimate the missing value for the fourth quarter of 2025, while approximation methods—including linear approximation, quadratic approximation, and cubic spline interpolation—were used to predict future RPPI values for the years 2026 and 2027.
